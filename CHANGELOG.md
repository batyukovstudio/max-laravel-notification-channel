# Changelog

## v1.0.0 - 2026-03-31

Первая публичная версия пакета Laravel Notification Channel для MAX API.

## 1.0.0 - 2026-03-31

- Replaced the previous messenger-specific package API with a MAX-focused implementation.
- Added MAX message delivery, uploads, updates, webhook subscriptions, callback answers, and message management helpers.
- Reorganized the internals using Porto SAP with `Containers` and `Ship`.
- Rewrote the test suite and documentation for the new MAX channel.
